# Just Google It

Have you ever thought that you have to learn something before start to learn something? Like let's say someone wants to learn coding and says to himself "I should learn math first. Is that just laziness? or something else? It's like driving a car in a circle forever and never reaching the destination. In this article I'll try to find that answer because why not?

First we need to clarify and simplify the issue so we can understand it better later. The problem is people often create unnecessary barriers before starting something new. They convince themselves they need to master prerequisite skills before tackling their main goal. But I think this just doesn't make sense. We should first start our main goal and then if we find something that we don't know, we can just google it and learn it.

Possible topics behind that I'll be covering in this article:

1. Preparation paralysis
2. Curse of Knowledge
3. Fear of Failure
4. Perfectionism
5. Imposter Syndrome
6. Self-Doubt
7. Strict linear path
8. Comfort Zone Attachment
